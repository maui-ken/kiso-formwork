# 基礎型枠 割付計算（PWA）

住宅基礎工事の型枠を、**各辺の長さ**と**手持ちの型枠在庫**から自動で割り付けるスマホ用アプリです。
依存ライブラリなし（HTML/CSS/JS のみ）。スマホのブラウザで開いて「ホーム画面に追加」すればアプリとして使えます（オフライン対応）。

## 機能（v1.0）
- **各辺の長さを手入力**（長方形は横×縦の一括入力も可）
- **型枠在庫の登録・管理**（直線枠／コーナー枠／調整枠）
- **割付計算**（職人の手順を再現）
  1. 各角にコーナー枠を配置
  2. 残り長さに、所有サイズの**大きい順**で直線枠を詰める
  3. 端の余りを**調整枠（伸縮）**または**板（任意長カット）**で埋める
- **両面（内・外）自動計算**（内側は基礎幅×2だけ短く計算）
- **必要数量 vs 在庫の過不足判定**（不足枚数・板カット寸法を表示）
- データは端末内（localStorage）に保存

## 計算の前提（v1.0の割り切り）
- 単位は **mm**。
- コーナー枠は「脚の長さ」を各辺から消費する1ユニットとしてモデル化（出隅/入隅の区別は未対応）。
- 内側の面の長さは `各辺 − 基礎幅×2` で近似（角の納まりは現場で微調整前提）。
- 直線枠の割付は **グリーディ（大きい順）**。職人の手順に一致。最小本数の厳密最適化は将来対応。

## 使い方
1. **型枠在庫**タブで、手持ちの枠（横幅mm・枚数）を登録。
2. **図面・寸法**タブで、現場名・基礎幅・各辺の長さを入力。
3. 「割付を計算する」→ **計算結果**タブで必要枚数と過不足を確認。

## ローカルで動かす
```bash
cd kiso-formwork
python3 -m http.server 4173
# ブラウザで http://localhost:4173 を開く
```

## テスト（計算エンジン）
macOS の JXA で実行：
```bash
osascript -l JavaScript test_engine.js
```

## 公開（リリース）方法
静的サイトなので、以下のいずれかにフォルダを上げるだけで公開できます（PWAはHTTPS必須）:
- **Netlify**: フォルダをドラッグ＆ドロップ
- **GitHub Pages**: リポジトリにpushして Pages を有効化
- **Vercel / Cloudflare Pages**: フォルダを連携

## ファイル構成
| ファイル | 役割 |
|---|---|
| `index.html` | 画面 |
| `styles.css` | スタイル |
| `engine.js` | 割付計算エンジン（純粋関数・テスト対象） |
| `app.js` | 状態管理・UI・永続化 |
| `manifest.webmanifest`, `sw.js`, `icon.svg` | PWA（インストール・オフライン） |
| `test_engine.js` | エンジンの自動テスト |
